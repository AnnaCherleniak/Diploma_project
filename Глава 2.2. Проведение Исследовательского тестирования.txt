Приступая к проведению Исследовательского тестирования, для этого в первую очередить нужно определиться с тем, какие туры буду использовать при тестировании.
Я выделила список из 13ти туров, которые подходят для тестирования моего веб-приложения:
1. Тур по путеводителю;
2. Тур по отметкам;
3. Интеллектуальный тур;
4. Тур службы доставки;
5. Тур уборщика;
6. Тур актера второго плана;
7. Тур по темным переулкам;
8. Тур супермодели;
9. Тур, отмененный из-за дождя;
10. Тур лежебоки;
11. Тур саботажника, диверсанта;
12. Антисоциальный тур;
13. Обсессивно-компульсивный тур или тур невротика;

Для более эффективного планирования тестирования составлена матрица тестирования(Приложение 1.)
Матрица тестирования — это основа для совершенствования. На вертикальной оси отображены районы и туры, которые используются в качестве плана тестирования. На горизонтальной оси - типы возможных ошибок, найденных при тестировании. Т.о. матрица позволяет убедится, что в результате проведенных проверок будут охвачены все типы ошибок, необходимые на данном этапе проверки продукта.
На данный момент (перед составлением Техннического Заддания) запланировано провести тестирование на выявленение след типов ошибок:
функциональьные;
лингвистические;
юзабилити;
интерфейсные.
-------------

    Как пользоваться методикой

Выбрать тур из списка ниже.
Изучить его цели.
Поставить таймер на 2 часа (час, полчаса).
Провести исследование системы строго по целям тура. Ни на что не отвлекаясь, только “миссия” тура.
При необходимости повторить.

* креш - программная ошибка, аварийное закрытие программы

1. Тур по путеводителю - функциональное тестирование.
2. Тур по отметкам - функциональное тестирование.
3. Интеллектуальный тур - функциональное тестирование.
4. Тур службы доставки - функциональное тестирование.
5. Тур уборщика - функциональное тестирование, юзабилити
6. Тур актера второго плана (The Supporting Actor Tour) - функц и юзабилити
7. Тур по темным переулкам (The Back Alley Tour) - юзабилити
8. Тур супермодели - юзабилити
9. Тур, отмененный из-за дождя - функциональное тестирование, юзабилити
10. Тур лежебоки - функциональное тестирование, юзабилити
11. Тур саботажника, диверсанта - функциональное тестирование
12. Антисоциальный тур - функциональное тестирование, юзабилити
13. Обсессивно-компульсивный тур или тур невротика - нефункциональные требованя
